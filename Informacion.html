# Formulario ¿Para que sirve? Operaciones que realiza
1 Articulos

Este formulario sirve para solicitar u obtener uno o varios registros,permitiendo asi acceder a la base de datos MysQL en la tabla Articulos para realizar operaciones como insertar , actualizar y eliminar.

Obtiene los registros de los articulos con el método GET.

Obtiene un registro de los articulos con el método POST.

Insertara un registro de Pedidos con el método POST.

Eliminara un registro existente de los articulos con el método DELETE.

Actualizara un registro de los articulos existente con el método PUT.
2 Facturas

Este formulario sirve para solicitar u obtener uno o varios registros,permitiendo asi acceder a la base de datos MysQL en la tabla Facturas para realizar operaciones como insertar , actualizar y eliminar.

Obtiene y consulta todos los registros mediante el método GET.

Obtiene y solicita un registro mediante el método POST.

Inserta un registro mediante el método POST.

Elimina un registro existente mediante el metodo DELETE.

Actualiza una o varias columnas de un registro existente mediante el metodo PUT.
3 Pedidos

Este formulario permite acceder a los registros almacenados en la base de datos MYSQL de la tabla Pedidos especificamente. y permite realizar las acciones de obtener los datos, insertar registros, actualizar y eliminar registros.

Obtiene los registros de Pedidos con el método GET.

Obtiene un registro de Pedidos con el método POST.

Insertara un registro de Pedidos con el método POST.

Eliminara un registro existente de Pedidos con el metodo DELETE.

Actualizara un registro de Pedidos existente con el metodo PUT.
4 Socios

Este formulario permite acceder a los registros almacenados en la base de datos MYSQL de la tabla Socios. y permite realizar las acciones de obtener los datos de los socios, insertar, actualizar y eliminar registros de socios.

Obtiene los registros de todos los socios registrados con el método GET.

Obtiene el registro de un socio en espesifico con el método POST.

Insertara un registro de socio con el método POST.

Eliminara un registro de un socio con el metodo DELETE.

Actualizara un registro de un socio en espesifico existente, con el metodo PUT.
